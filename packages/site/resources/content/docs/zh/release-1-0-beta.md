# Gravito 1.0 Beta

Gravito 1.0 Beta 以 Bun 原生效能為核心，提供完整的全端開發體驗。本版本聚焦於
執行效能、統一的應用生態、融合 Prisma/Drizzle 設計優勢的原生 ORM，以及企業級
架構能力。

## 重點亮點

- Bun-native 架構：極快啟動與高效能執行
- 全端生態：一致的開發流程與工具鏈
- 原生 ORM：吸收 Prisma/Drizzle 的設計優勢，提供現代化資料模型
- **Atlas 官網上線**：前往 [atlas.gravito.dev](https://atlas.gravito.dev) 探索 Atlas ORM 的完整功能與即時演示。
- 企業級架構：可擴充、可演進的分層設計
- 可觀測性：標準化健康檢查、監控與追蹤能力

## 已發佈 Beta 套件

- @gravito/enterprise@1.0.0-beta.1
- @gravito/launchpad@1.0.0-beta.1
- @gravito/spectrum@1.0.1-beta.1

## 注意事項

- 此版本為 Beta，API 仍可能調整
- 建議先於 staging 環境驗證後再進入生產
