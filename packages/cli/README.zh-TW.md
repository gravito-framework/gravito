# Orbit Pulse

> Gravito 官方 CLI，提供專案建立、程式碼產生與開發工具。

## 安裝

```bash
# 全域安裝（建議）
bun add -g @gravito/pulse

# 或使用 bunx
bunx @gravito/pulse create my-app
```

## 快速開始

```bash
# 互動式建立專案
gravito create my-app

# 使用模板
gravito create my-app --template basic
```

## 常用指令

```bash
# 產生 Controller
gravito make:controller UserController

# 列出路由
gravito route:list
```
